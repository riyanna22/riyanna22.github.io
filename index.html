GitHub Pages

Create Github Pages using the Git System:
  1. Login GitHub account and click the new repository
  2. Create a new repository with the name riyanna22.github.io, be sure it is public
  3. Doing git bash on the created folder
  4. Make this folder a local repository with git init
  5. Connect local repository to github repository with git remote
  6. Then git add to add all new files to the repository
  7. git commit –m “first commit” which means that the first commit we did
  8. Enter git push -u origin master to send the file changes, Wait until all files are finished uploading
